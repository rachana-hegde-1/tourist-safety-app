# App Icons for Tourist Safety PWA

This directory contains app icons for the Progressive Web App.

## Icon Requirements:
- 72x72, 96x96, 128x128, 144x144, 152x152, 192x192, 384x384, 512x512
- Shield design with location pin
- PNG format with transparency
- High contrast for visibility

## Icon Description:
The main app icon should be a shield shape with a location pin in the center,
representing safety and location tracking for tourists.

## Colors:
- Shield: Blue gradient (#1D9E75 to #3B82F6)
- Location pin: Red (#DC2626) for emergency/safety emphasis
- Background: Transparent

## Usage:
These icons are referenced in the manifest.json file and will be used for:
- Home screen icons
- App launcher icons
- Browser favicons
- Splash screen icons

## To Generate:
Use a design tool like Figma, Adobe Illustrator, or Canva to create
the actual PNG files with the specifications above.
